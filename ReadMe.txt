
                   AEG Mesher
                   ==========

Octave/MATLAB Structured Mesh Generator 
=======================================


Authors
-------

I. D. Flintoft  ian.flintoft@york.ac.uk
M. Berens       michael-berens1@web.de
J. F. Dawson    john.dawson@york.ac.uk


Features
--------

Conversion between AMELET-HDF [1], CONCEPT-II [2] and Gmsh [3]
unstructured mesh formats.

Structured mesh generator for cubic, uniform and non-uniform 
(incomplete) cuboid meshes.

Structured mesh exporter for UoY AEG Vulture FDTD code [4].

are also relevant to reporting bugs.
